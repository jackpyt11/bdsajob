# **App Name**: ChakriBarta

## Core Features:

- Job Circular Display: View active job circulars with details like position, salary, and application deadline, ensuring applicants can quickly find relevant opportunities.
- Smart Online Application Form: A guided, step-by-step application form with auto-validation for critical fields (e.g., mobile number, email format) to ensure data accuracy and improve user experience.
- AI-Assisted Photo & Signature Tool: Applicants can upload images, and the system, powered by AI, acts as a tool to analyze and intelligently suggest optimal resizing, cropping, or compression parameters to meet specified dimensions and file size limits.
- Secure Payment & Unique User ID: Upon form submission, a unique user ID is auto-generated for the applicant. The system integrates directly with popular payment gateways (bKash, Nagad, Rocket) via pop-up for seamless payment processing.
- Auto-Generated PDF Documents: After successful payment, applicants can immediately download a 'PAID' stamped PDF invoice/applicant copy. Later, they can log in to download their admit card upon announcement of exam dates.
- Admin Dashboard & Circular Management: Administrators get an overview of application statistics (total, paid, unpaid) and can manage job circulars—adding new ones, editing existing details, or toggling their active status.
- Applicant Data Management & Export: Admin panel features advanced search and filter options for applicant data using User ID or mobile number. Payment verification logs are available, and applicant lists can be exported to Excel/CSV.

## Style Guidelines:

- The visual design draws inspiration from concepts of professionalism, clarity, and efficiency, aligning with official application processes. The color scheme is light to enhance readability.
- Primary color: A deep, professional blue (#216BE3), chosen for its association with reliability and trust, making it suitable for clear calls to action and main headings.
- Background color: A very light, cool blue-grey (#E9EEF5), providing a clean, spacious backdrop that aids focus on content and forms.
- Accent color: A moderate, slightly muted purple (#9969DA), offering a pleasant contrast to the primary blue and can be used for secondary elements or highlight information without being overwhelming.
- Body and headline font: 'Inter', a grotesque-style sans-serif, selected for its modern, machined, and neutral aesthetic, ensuring excellent readability across various text lengths in a professional context.
- Use clear, intuitive icons with a consistent line-style throughout the application for easy navigation and understanding of different functionalities and administrative tasks.
- Employ a clean, responsive layout across all views. Multi-step forms will feature prominent progress indicators, and admin dashboards will be organized using structured cards and tables for clarity.
- Incorporate subtle and purposeful animations for user feedback, such as confirmation after form submission, loading states, and state changes, ensuring they are non-distracting and enhance usability.
